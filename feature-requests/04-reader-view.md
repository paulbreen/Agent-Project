# Feature: Reader View

**User Story:**

As a user, I want to be able to read a saved article in a clean, distraction-free format.

**Acceptance Criteria:**

*   When a user opens a saved article, it should be displayed in a "reader mode".
*   The reader mode should have a clean, readable font and layout.
*   The user should be able to adjust the font size and switch between light and dark themes.
*   The reader view should display the article's title and content.
